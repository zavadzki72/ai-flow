# Git Workflow

- **Branch principal:** `develop`
- **Nomenclatura de branches:** `feature/descricao`, `fix/descricao`, `chore/descricao`
- **Commit convention:** Conventional Commits (`feat:`, `fix:`, `chore:`, `docs:`, `test:`)
- **Pull Request:** base em `develop`, ao menos 1 aprovação
- **Commit por etapa:** cada baby step do PLAN deve ter seu próprio commit
